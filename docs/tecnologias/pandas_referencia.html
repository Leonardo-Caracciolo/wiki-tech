pandas.

Métodos esenciales organizados por categoría, con ejemplos de input y output.

Limpieza Lectura Filtrado Merge / Join Agrupación Columnas Filas Exportar

Limpieza

.str.strip() Elimina espacios (u otros caracteres) al inicio y al final
Input
# " Spotify " → "Spotify" df["Cliente"].str.strip() # lstrip() → solo izquierda # rstrip() → solo derecha # strip("x") → elimina el carácter "x"
Output
0 Spotify 1 Verizon 2 PPG
.str.replace() Reemplaza substrings o patrones regex dentro de strings
Input
# Eliminar dobles espacios df["col"].str.replace(r"\s+", " ", regex=True) # Reemplazar texto literal df["col"].str.replace("S.A.", "", regex=False)
Output
# "Hola mundo" → "Hola mundo" # "Google S.A." → "Google"
Argumentos
patString o regex a buscar
replString de reemplazo
regex=TrueSi es False, trata pat como literal
case=TrueSi es False, ignora mayúsculas
.fillna() Rellena valores NaN con un valor específico
Input
# Con valor fijo df["Cuit"].fillna("Sin CUIT") # Con el valor anterior (forward fill) df["col"].fillna(method="ffill") # Con la media de la columna df["monto"].fillna(df["monto"].mean())
Output
0 30-71757936-0 1 Sin CUIT # NaN → "Sin CUIT" 2 30-68651307-2
Argumentos
valueValor fijo para reemplazar NaN
method"ffill" (anterior) / "bfill" (siguiente)
limitMáximo de NaN consecutivos a rellenar
.dropna() Elimina filas o columnas que contienen NaN
Input
# Elimina filas con CUALQUIER NaN df.dropna() # Solo si TODAS las columnas son NaN df.dropna(how="all") # Solo revisando ciertas columnas df.dropna(subset=["Cuit", "Cliente"])
Output — antes vs después
# ID Cliente Cuit # 1 Spotify 30-71... ✅ # 2 NaN NaN ❌ eliminada # 3 PPG 30-68... ✅
Argumentos
axis=00 = filas, 1 = columnas
how="any""any" = al menos 1 NaN / "all" = todos NaN
subsetLista de columnas a revisar
.drop_duplicates() Elimina filas duplicadas
Input
# Duplicados en todas las columnas df.drop_duplicates() # Duplicados solo por ID_Cliente df.drop_duplicates(subset=["ID_Cliente"]) # Mantener el último en vez del primero df.drop_duplicates(subset=["ID_Cliente"], keep="last")
Argumentos
subsetColumnas a considerar para comparar
keep="first""first" / "last" / False (elimina todos)
.astype() Convierte el tipo de dato de una columna
Input
df["ID_Cliente"].astype(str) # int → string df["monto"].astype(float) # str → float df["activo"].astype(bool) # int → bool
Output
# dtype: int64 → dtype: object (str) # dtype: object → dtype: float64
⚠️ Si hay NaN en la columna, el casteo a int falla. Usar fillna() antes o usar Int64 (nullable).

Lectura e Inspección

.head() / .tail() Muestra las primeras o últimas N filas
Input
df.head(3) # primeras 3 filas (default=5) df.tail(2) # últimas 2 filas
Output
ID_Cliente Cliente Cuit 0 1 Spotify 30-71... 1 2 Verizon 30-70... 2 3 PPG 30-68...
.info() Resumen del DF: tipos, NaN count, memoria
Input
df.info()
Output
RangeIndex: 3 entries # Column Non-Null Dtype --- ---------- -------- ----- 0 ID_Cliente 3 non-null int64 1 Cliente 3 non-null object 2 Cuit 2 non-null object ← tiene NaN
.describe() Estadísticas descriptivas de columnas numéricas
Input
df.describe() # Para incluir columnas de texto también: df.describe(include="all")
Output
ID_Cliente count 3.0 mean 2.0 std 1.0 min 1.0 25% 1.5 50% 2.0 75% 2.5 max 3.0
.value_counts() Cuenta ocurrencias únicas de cada valor
Input
df["Herramienta"].value_counts() # Incluir NaN: df["Herramienta"].value_counts(dropna=False) # En porcentaje: df["Herramienta"].value_counts(normalize=True)
Output
NFE Alert 5 Acrobat 3 SAP 2
Argumentos
normalize=FalseTrue → devuelve proporciones (0 a 1)
sort=TrueOrdena de mayor a menor
ascending=FalseTrue → orden ascendente
dropna=TrueFalse → incluye NaN en el conteo

Filtrado y Selección

df[ condición ] Filtra filas según una condición booleana
Input
# Igualdad df[df["Usa Herramienta"] == "SI"] # Múltiples condiciones (& = AND, | = OR) df[(df["Usa Herramienta"] == "SI") & (df["ID_Cliente"] == 1)] # Negación df[df["Cuit"].notna()]
⚠️ Las condiciones múltiples van entre paréntesis y usan & / | (no and / or).
.loc[ ] / .iloc[ ] Selección por etiqueta (loc) o por posición numérica (iloc)
Input
# loc: por nombre de índice/columna df.loc[0, "Cliente"] # → "Spotify" df.loc[0:2, ["Cliente", "Cuit"]] # filas 0-2, 2 columnas # iloc: por posición entera df.iloc[0, 1] # fila 0, columna 1 df.iloc[:3, :2] # primeras 3 filas, 2 columnas
.isin() Filtra valores que estén dentro de una lista
Input
clientes_vip = [1, 3] df[df["ID_Cliente"].isin(clientes_vip)] # Negación (los que NO están) df[~df["ID_Cliente"].isin(clientes_vip)]
Output
# Solo filas con ID 1 y 3 ID_Cliente Cliente 0 1 Spotify 2 3 PPG
.query() Filtra con una expresión en string (más legible)
Input
df.query("ID_Cliente == 1 and Usa_Herramienta == 'SI'") # Con variable externa (usar @) minimo = 2 df.query("ID_Cliente >= @minimo")
💡 Las columnas con espacios en el nombre hay que escaparlas con backticks: `Usa Herramienta`.

Merge / Join

.merge() Une dos DataFrames por una o más columnas clave
Input
df_herramientas.merge( df_clientes[["ID_Cliente", "Cuit"]], on="ID_Cliente", how="left" ) # Claves con distinto nombre en cada DF: df_a.merge(df_b, left_on="id", right_on="cliente_id")
Argumentos — how=
"left"Todas las filas del DF izquierdo. NaN donde no hay match a la derecha
"right"Todas las filas del DF derecho. NaN donde no hay match a la izquierda
"inner"Solo filas con match en ambos (default SQL)
"outer"Todas las filas de ambos lados. NaN donde no hay match
Más argumentos
onColumna(s) clave si tienen el mismo nombre en ambos DFs
left_on / right_onClaves cuando los nombres difieren entre DFs
suffixes=("_x","_y")Sufijos para columnas duplicadas. "" = sin sufijo
pd.concat() Apila DataFrames vertical u horizontalmente
Input
# Apilar filas (vertical) pd.concat([df_enero, df_febrero, df_marzo]) # Apilar columnas (horizontal) pd.concat([df_a, df_b], axis=1) # Resetear índice después de apilar pd.concat([df1, df2], ignore_index=True)
Argumentos
axis=00 = apilar filas (vertical), 1 = columnas (horizontal)
ignore_index=FalseTrue → resetea el índice del resultado
join="outer""outer" = incluye todas las columnas, "inner" = solo las comunes

Agrupación y Agregación

.groupby() Agrupa el DF por una columna para aplicar funciones de agregación
Input
# Contar herramientas por cliente df.groupby("ID_Cliente")["Herramienta"].count() # Suma por grupo df.groupby("ID_Cliente")["Monto"].sum() # Múltiples agregaciones a la vez df.groupby("ID_Cliente").agg({ "Herramienta": "count", "Monto": ["sum", "mean"] })
Output — count por cliente
ID_Cliente 1 3 2 2 3 4
.pivot_table() Tabla dinámica: agrupa, agrega y pivotea en una sola operación
Input
df.pivot_table( values="Monto", index="Cliente", columns="Herramienta", aggfunc="sum", fill_value=0 )
Output
Herramienta Acrobat NFE Alert SAP Cliente PPG 100 200 0 Spotify 0 500 300
Argumentos
valuesColumna a agregar
indexColumna que se convierte en filas
columnsColumna que se convierte en columnas
aggfunc"sum", "mean", "count", "max", etc.
fill_valueValor para NaN resultantes (ej: 0)
.sort_values() Ordena el DF por una o más columnas
Input
# Ascendente por Cliente df.sort_values("Cliente") # Descendente df.sort_values("Monto", ascending=False) # Múltiples columnas df.sort_values(["ID_Cliente", "Herramienta"], ascending=[True, False])
Argumentos
byColumna o lista de columnas
ascending=TrueFalse = descendente. Lista si hay múltiples columnas
na_position="last""first" → NaN al inicio

Manipulación de Columnas

.rename() Renombra columnas o índices
Input
df.rename(columns={ "ID_Cliente": "id", "Usa Herramienta": "activo" })
Output — columnas renombradas
# Antes: ["ID_Cliente", "Cliente", "Usa Herramienta"] # Después: ["id", "Cliente", "activo"]
.drop() Elimina columnas o filas por nombre
Input
# Eliminar columnas df.drop(columns=["Cuit_formula", "col_extra"]) # Eliminar por índice de fila df.drop(index=[0, 2])
Argumentos
columnsLista de columnas a eliminar
indexLista de índices de filas a eliminar
inplace=FalseTrue → modifica el DF sin reasignar
.assign() Crea o sobreescribe columnas sin modificar el original
Input
df.assign( Cuit_limpio=df["Cuit"].str.strip(), tiene_cuit=df["Cuit"].notna() ) # Equivalente a: df["Cuit_limpio"] = df["Cuit"].str.strip()
💡 .assign() devuelve una copia. Ideal para encadenar transformaciones sin perder el DF original.
.apply() Aplica una función a cada fila o columna
Input
# A cada valor de una columna df["Cliente"].apply(lambda x: x.upper()) # A cada fila (axis=1) df.apply(lambda row: row["Cliente"] + " - " + row["Cuit"], axis=1) # A todas las columnas string df.apply(lambda col: col.str.strip() if col.dtype == "object" else col)
Argumentos
funcFunción o lambda a aplicar
axis=00 = por columna, 1 = por fila

Manipulación de Filas

.reset_index() Resetea el índice del DF (útil después de filtrar o concatenar)
Input
df_filtrado = df[df["Usa Herramienta"] == "SI"] df_filtrado.reset_index(drop=True)
Output
# Antes: índice 0, 2, 5, 8 (después de filtrar) # Después: índice 0, 1, 2, 3 (secuencial limpio)
Argumentos
drop=FalseTrue → descarta el índice viejo. False → lo convierte en columna
.sample() Devuelve una muestra aleatoria de filas
Input
# 5 filas aleatorias df.sample(5) # 10% del total df.sample(frac=0.1) # Con semilla para reproducibilidad df.sample(5, random_state=42)
Argumentos
nNúmero de filas a devolver
fracFracción del total (ej: 0.2 = 20%)
random_stateSemilla para resultados reproducibles
.iterrows() Itera fila por fila (cuando no queda otra opción)
Input
for idx, row in df.iterrows(): print(idx, row["Cliente"], row["Cuit"])
⚠️ iterrows() es lento en DFs grandes. Siempre preferir operaciones vectorizadas (.apply, máscaras booleanas). Usarlo solo si la lógica es muy compleja.

Exportar

.to_excel() Exporta a un archivo .xlsx
Input
# Simple df.to_excel("output.xlsx", index=False) # Múltiples sheets en un mismo archivo with pd.ExcelWriter("output.xlsx", engine="openpyxl") as writer: df_clientes.to_excel(writer, sheet_name="Clientes", index=False) df_herr.to_excel(writer, sheet_name="Herramientas", index=False)
Argumentos
index=TrueFalse → no escribe el índice como columna
sheet_nameNombre de la hoja en el Excel
startrow / startcolCelda de inicio (útil para dejar espacio)
.to_csv() Exporta a CSV
Input
df.to_csv("output.csv", index=False, encoding="utf-8-sig") # utf-8-sig: agrega BOM para que Excel lo abra bien # sep=";" para separador punto y coma (europeo) df.to_csv("output.csv", sep=";", index=False)
Argumentos
sep=","Separador de columnas
encoding"utf-8", "utf-8-sig", "latin-1"
decimal=".""," para usar coma decimal (ES)
pd.read_excel() / pd.read_csv() Lee archivos externos como DataFrame
Input
# Excel df = pd.read_excel("datos.xlsx", sheet_name="Clientes", engine="openpyxl") # CSV df = pd.read_csv("datos.csv", sep=";", encoding="latin-1") # Leer solo algunas columnas df = pd.read_excel("datos.xlsx", usecols=["ID_Cliente", "Cuit"])
Argumentos clave
sheet_nameNombre o índice de la hoja (Excel). 0 = primera hoja
usecolsLista de columnas a leer (optimiza memoria)
skiprowsFilas a saltear al inicio (ej: encabezados extras)
nrowsLímite de filas a leer
dtypeDict para forzar tipos: {"ID": str, "Monto": float}
